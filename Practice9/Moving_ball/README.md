# Moving Ball Game

A Pygame game where you control a red ball using arrow keys.

## Features
- Red ball (radius 25) on a light grid background
- Arrow key movement, 20px per keypress
- Ball cannot leave screen boundaries
- Displays current position in real-time

## Run
```bash
pip install pygame
python main.py
```

## Controls
| Key        | Action     |
|------------|------------|
| Arrow Up   | Move up    |
| Arrow Down | Move down  |
| Arrow Left | Move left  |
| Arrow Right| Move right |
| ESC        | Quit       |

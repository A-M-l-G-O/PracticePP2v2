# Music Player

A Pygame-based interactive music player with keyboard controls.

## Features
- Keyboard controls: P (Play), S (Stop), N (Next), B (Back), Q (Quit)
- Displays current track name, status, elapsed time, and track index
- Supports MP3, WAV, OGG files

## Run
```bash
pip install pygame
python main.py
```

## Adding Music
Place your `.mp3`, `.wav`, or `.ogg` files inside the `music/` folder.
The player will automatically detect and list them alphabetically.

## Controls
| Key | Action        |
|-----|---------------|
| P   | Play current  |
| S   | Stop          |
| N   | Next track    |
| B   | Previous      |
| Q   | Quit          |

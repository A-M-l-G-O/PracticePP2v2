# Mickey's Clock

A Pygame-based analog clock using Mickey Mouse-style hand graphics.

## Features
- Displays current system time (minutes and seconds)
- Right hand = minutes, Left hand = seconds
- Real-time sync, updates every second

## Run
```bash
pip install pygame
python main.py
```

## Notes
- Place a `mickey_hand.png` inside `images/` for a custom hand graphic.
- If the image is missing, a default hand is drawn automatically.
- Press `ESC` to quit.
